Menu bar app that bridges the browser to Blender.

Setup

  1. Open the rrelay disk image, drag the rrelay app to your Applications folder, and launch it. It lives in the menu bar.
  2. Click the menu bar icon and press Start.
  3. In Blender, open Edit → Preferences → Add-ons → Install, pick the marionettes zip, then enable it.
  4. Open the marionettes tab in the 3D viewport's N panel and press Start.
  5. Back here, open an interface and click start camera.
